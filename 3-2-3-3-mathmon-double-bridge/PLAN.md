# 매스몬 두 배 다리 제작 계획 (3-2-3-3)

## Context
- 3학년 2학기 3단원 「원」 3차시. 영역: 도형과 측정. 성취기준(안내용): 지름·반지름 관계 — 고시 원문 대조 필요.
- 배움주제: 지름 = 반지름 × 2. 학생 목표: 반지름↔지름을 두 배·반으로 바꾼다.

## 수학 설계
- 한 문제는 한 방향만: 반지름→지름(×2) 또는 지름→반지름(÷2). 한 판 10문제, 반지름 2~8cm.
- 생성기: `buildDoubleProblem()`. 정답 = 변환값. 보기 = 정답 + **핵심 오답**(반지름→지름이면 "두 배 잊음"=r, 지름→반지름이면 "반 안 나눔"=2r) + 근접 오답. 작은 값에서 겹치면 채움으로 4개 distinct 보장.
- 항등: `diameter === radius * 2` (스크립트 20000회 검증).

## 단계 선택 (핵심)
- 1단계. 다리 그림: 위=반지름 칸, 아래=지름 칸(반지름 두 칸, 가운데 divider). 주어진 칸은 채움, 묻는 칸은 점선+"?".
- 정답 1개 + 핵심 오답 + 근접. 오답 2회 → 공개("정답은 N cm").

## 보상/결과 (rocket 엔진 라벨만 교체)
