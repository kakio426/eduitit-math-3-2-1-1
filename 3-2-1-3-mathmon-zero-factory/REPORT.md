# 매스몬 0 공장 설명 보고서

## 1. 개요

`매스몬 0 공장`은 3학년 2학기 1단원 3차시에서 다루는 `(몇십)×(몇십), (몇십몇)×(몇십)` 계산을 짧은 게임 흐름으로 연습하는 수학 게임입니다. 학생은 곱셈 문제를 보고 0을 잠깐 떼어 낸 핵심 곱을 먼저 고른 뒤, 끝에 다시 붙일 0의 개수를 골라 완성 수를 만듭니다.

핵심 목표는 `핵심 부분만 곱한 뒤, 두 수 끝의 0 개수를 합쳐 답 뒤에 붙이는 원리`를 자연스럽게 반복하게 만드는 것입니다.

## 2. 학습 설계

- 문제 유형: `(A0)×(B0)`, `(AB)×(C0)`
- 문제 은행: 두 유형에서 매 판 5문제씩 섞어 총 10문제 출제
- 입력 방식: 1단계 핵심 곱 선택, 2단계 0 개수 선택
- 피드백: 첫 오답에는 짧은 힌트, 두 번째 오답에는 정답과 설명을 보여 준 뒤 다음 단계로 이동
- 대표 오개념 처리: 2단계 선택지에는 0을 한 개 적게 붙이거나 한 개 많이 붙이는 보기가 항상 들어감
- 보상: 한 문제 완료마다 출하 준비 칸 1칸과 생산 이벤트 1회 적용. 정답 문제는 보통 생산, 불량 발생, 대량 생산, 즉시 출하, 컨베이어 정지, 무지개 부품 중 하나가 나오고, 오답이 있었던 문제는 불량 처리만 적용. 보상 팝업은 생산·불량·정지·무지개 상태를 서로 다른 공장 부품 아이콘으로 보여 줌
- 출하 등급: 생산량 1~100%를 측정해 우리 반 -> 학교 -> 동네 -> 도시 -> 전국 -> 세계 출하를 공개
- 특별 단계: 무지개 부품을 한 번 얻으면 일반 생산량보다 우주 특별 출하가 우선
- 최종 보상: 출하 등급과 정답 수 조건을 함께 보아 0 공장 매스몬 카드 1장을 공개하고, 카드 다운로드를 생성함

## 3. 게임 흐름

```text
첫 화면 -> 설명 화면 -> 핵심 곱 선택 -> 0 개수 선택 -> 컨베이어 완성 -> 생산 이벤트 -> 다음 문제 또는 즉시 출하 -> 생산량 측정 -> 출하 등급 + 매스몬 카드
```

학생은 예를 들어 `30 × 40`에서 먼저 `3 × 4 = 12`를 고릅니다. 다음 단계에서 두 수 끝에 있던 0이 모두 2개였다는 것을 골라, 컨베이어 보드에 0 스티커 2개가 붙고 `1,200`이 완성됩니다.

`27 × 40`에서는 `27 × 4 = 108`을 고른 뒤 0을 1개 붙여 `1,080`을 완성합니다. 이처럼 문제마다 통째 계산보다 `핵심 곱`과 `0 개수`가 먼저 보이도록 설계했습니다.

## 4. 화면별 설명

### 첫 화면

첫 화면은 `cover-generated.webp`를 RasterStage 배경으로 사용합니다. 0 공장과 컨베이어가 보이는 대표 장면 위에 게임 제목, 한 줄 목표, 시작 버튼을 HTML로 얹습니다.

### 설명 화면

설명 화면은 3단계만 보여 줍니다.

1. 끝의 0을 잠깐 떼고 핵심 곱을 계산합니다.
2. 떼어 낸 0의 개수를 세어 다시 붙입니다.
3. 완성한 물건을 컨베이어로 출하합니다.

버튼 문구는 다음 행동이 바로 보이도록 `생산 시작`으로 둡니다.

### 문제 화면

문제 화면은 왼쪽에 0 공장 생산 상태와 10칸 출하 준비 칸, 오른쪽에 문제·컨베이어 보드·선택지를 둡니다. 컨베이어 보드는 `핵심 곱`, `0 스티커`, `완성 수` 세 칸으로 구성됩니다. 1단계를 맞히면 핵심 곱 칸이 채워지고, 2단계를 맞히면 0 스티커가 실제 개수만큼 나타난 뒤 완성 수가 자동으로 채워집니다.

출하 준비 칸은 문제마다 1칸씩 채워집니다. 한 번에 맞힌 문제는 노란 완성품 칸으로 남고, 중간에 틀린 문제는 빨간 불량 처리 칸으로 남아 학생이 자신의 진행을 숫자만이 아니라 공장 상태로 볼 수 있게 했습니다.

### 보상 화면

한 문제의 두 단계가 끝나면 화면 중앙에 생산 이벤트 결과가 뜹니다. 보통은 생산량이 늘지만, 대량 생산으로 크게 늘거나, 불량 발생으로 줄거나, 컨베이어가 멈춰 0이 되거나, 즉시 출하가 열리거나, 무지개 부품이 나올 수 있습니다. 문제 안에서 한 번이라도 틀리면 `불량 처리`가 적용됩니다. 팝업에는 이벤트 종류에 맞는 부품 아이콘을 크게 보여 주어 1차시 상자 보상처럼 보상 상태가 바로 읽히게 했습니다.

### 결과 화면

결과 화면은 출하 장면 RasterStage 배경 위에 생산량 측정 막대, 출하 등급 트랙, 정답 수, 매스몬 카드를 HTML로 얹습니다. 생산량은 일반 등급을 정하고, 무지개 부품은 우주 특별 출하를 엽니다. 매스몬 카드는 생산량뿐 아니라 정답 수 조건도 함께 보아 공개됩니다. 생산량이 0이면 별도의 컨베이어 점검 배경을 사용해 성공 결과와 재도전 결과가 시각적으로 구분됩니다.

## 5. 매스몬 역할

3차시의 매스몬은 1차시 기준 공용 `base-pack` 10종 중 이번 판 결과에 맞는 카드 1장으로 공개됩니다. 0 공장의 중심 보상은 생산량과 출하 등급이고, 매스몬 카드는 그 결과를 기념하는 보상입니다.

현재 기준은 1차시 `매스몬 상자런`의 밝은 2D 애니/스티커형 매스몬 톤입니다. 원본 관리는 `_shared/mathmon/base-pack/`에서 하고, 3차시 실행 패키지에는 `assets/mathmon/base-pack/*.webp` 배포본을 복사했습니다. 기존 0 공장팩과 V2 장난감/클레이풍 팩은 보존만 하고 실행에는 쓰지 않습니다.
