## Static Game Sharing Via GitHub Pages

학생용 정적 HTML 게임은 이 레포지토리의 GitHub Pages 설정으로 공유한다. 필요한 게임 폴더만 `actions/upload-pages-artifact`와 `actions/deploy-pages`로 배포하고, 공유 전 `curl -I -L <url>`로 실제 URL이 `HTTP 200`인지 확인한다. Pages 실패 시 `gh run view <run-id> --log-failed`로 원인을 먼저 확인한다. checkout 단계에서 submodule 오류가 나면 `.gitmodules`와 `git ls-files -s`로 깨진 `160000` gitlink를 확인하고, 필요 없는 임시 gitlink는 `git rm <path>`로 제거한다.
