# 매스몬 원 무늬 디자이너 제작 계획 (3-2-3-4) ★ 단원 정점

## Context
- 3학년 2학기 3단원 「원」 4차시(단원 정점). 영역: 도형과 측정. 성취기준(안내용): 원으로 규칙·무늬 — 고시 원문 대조 필요.
- 배움주제: 원으로 규칙·무늬 만들기. 학생 목표: 같은 크기 원을 규칙대로 이어 무늬를 만든다.
- 정점 소환: 3-1(중심·반지름·지름)·3-2(컴퍼스 등반지름)·3-3(두 배 관계)를 "같은 크기 원을 규칙대로 배치"로 종합.

## 수학 설계
- 한 문제: 같은 반지름 원이 같은 간격으로 일부 놓인 무늬 + 빈 자리 후보(가·나·다·라). 다음 자리를 고른다. 한 판 10문제.
- 생성기: `buildPatternProblem()`. 반지름 14~18, 간격 = 2r+12~18, 놓인 원 2~3개.
- 후보 4개: 정답(같은 줄·같은 간격·같은 크기) + 오답 3종(너무 멂 `far` / 줄 벗어남 `off` / 크기 깨짐 `big`). 정답 정확히 1개(스크립트 20000회 검증, 모두 보드 안).

## 단계 선택 (핵심)
- 1단계. 놓인 원(파랑 채움) + 후보 원(점선 링, 가나다라) + 안내선. 정답 클릭 시 민트, 오답 로즈.
- 오답 2회 → 공개.
