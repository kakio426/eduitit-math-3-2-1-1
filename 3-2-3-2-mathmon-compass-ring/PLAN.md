# 매스몬 컴퍼스 마법진 제작 계획 (3-2-3-2)

## Context
- 3학년 2학기 3단원 「원」 2차시. 영역: 도형과 측정. 성취기준(안내용): 컴퍼스로 원 그리기 — 고시 원문 대조 필요.
- 배움주제: 컴퍼스로 원 그리기(반지름 = 컴퍼스 벌림). 학생 목표: 반지름에 맞게 컴퍼스를 벌린다.

## 수학 설계
- 한 문제: "반지름 N cm 원 그리기" → 컴퍼스 벌림(cm)을 고른다. 한 판 10문제, 반지름 2~6cm.
- 생성기: `buildCompassProblem(radius)`. 정답 = 반지름. 보기 = 반지름·**지름(2r) 함정**·한 칸 어긋남(±1/±2). 4개 distinct, 정답·지름 함정 항상 포함(스크립트 검증).
- 대표 오개념: **벌림을 지름으로 착각**(반지름 4 → 8 고르기). hint가 "지름이 아니라 반지름만큼"을 짚음.

## 단계 선택 (핵심)
- 1단계. 눈금(0~12) + 컴퍼스 V(0→r) + "반지름 r cm" 밴드를 보고 cm 값을 고름.
- 정답 1개 + 지름 함정 + 근접 오답. 오답 2회 → 정답 공개("정답은 r cm").

## 보상/결과 (rocket 엔진 라벨만 교체)
