# 매스몬 표적 맞히기

에듀잇티 수학 게임 시리즈 3단원(원) 1차시입니다.

- 대상: 3학년 2학기 3단원 1차시
- 학습: 원의 중심·반지름·지름 찾기
- 문제: 원 그림에서 한 용어(중심/반지름/지름)를 고르는 문제 10개 랜덤 출제
- 방식: 원 SVG 그림에서 후보 4개(가·나·다·라) 중 묻는 요소를 고르는 1단계 선택형
- 보상: 한 문제를 끝낼 때마다 랜덤 표적 점수 이벤트가 일어남(증가/감소/0/정중앙/무지개)
- 결과: 10발이 끝나면 표적 점수를 측정해 `빗나감 → 가장자리 → 명중 → 정중앙 → 표적왕` 중 어디까지 갔는지 보여 줌. 무지개 점수면 `전설 명중`. 점수가 0이면 다시하기
- 실행: `index.html`을 브라우저에서 열기

## 설계 의도

`매스몬 표적 맞히기`는 원의 세 구성 요소(중심·반지름·지름)를 **보이는 그림에서 직접 골라** 익히게 합니다. 한 문제는 한 용어만 묻고, 원 그림에는 정답 1개와 그 차시 핵심 오개념(반지름↔지름 혼동, 중심을 지나지 않는 현, 테두리 위 점)을 함께 보여 줍니다. 학생은 설명을 읽는 대신 그림에서 곧은 선·점을 눌러 답합니다.

한 문제를 끝낼 때마다 표적 점수에 일이 일어납니다. 보통은 점수가 오르지만, 가끔 크게 오르거나, 줄거나, 0이 되거나, 정중앙으로 직행하거나, 무지개 점수가 켜집니다. 문제 안에서 한 번이라도 틀리면 누수 이벤트로 처리합니다. 점수는 푸는 동안 전면에 두지 않고 한 판이 끝난 뒤 측정해 등급으로 보여 줍니다. `표적왕`은 어려운 최상위, `전설 명중`은 그보다 보기 힘든 secret 등급입니다.

## 시리즈 정체성

- 화면 골격(첫 화면 → 설명 → 문제 → 보상 → 결과), 상단 배지, 소리 버튼, 16:10/1280×800 Stage 계약은 시리즈 공통입니다.
- 문제 화면 도형은 생성 이미지가 아니라 **SVG**로 그립니다(읽기 쉬운 UI 우선).
